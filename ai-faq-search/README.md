# AI FAQ Semantic Search System

## Overview
This project is an AI-based semantic search system that retrieves relevant answers for user questions using vector embeddings.

## Features
- Semantic question answering
- Fast retrieval
- Dataset-based search
- Similarity scoring

## Dataset
The dataset contains frequently asked questions and answers.

## How it Works
1. Convert FAQ data into vector embeddings
2. Store embeddings (simulating Endee)
3. Convert user query into embedding
4. Perform similarity search
5. Return best matching answer

## How to Run
pip install sentence-transformers numpy  
python rag_app.py
